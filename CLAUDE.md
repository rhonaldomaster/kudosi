# Kudos Slack App

## Project Overview
Slack app for employee recognition (kudos) at Koombea.

## Tech Stack
- Node.js + Bolt for Slack
- PostgreSQL
- Railway (production)

## Commands
- `npm run dev` - Development with nodemon
- `npm start` - Production

## Project Structure
- `src/app.js` - Entry point
- `src/commands/` - Slash command handlers
- `src/views/` - Slack Block Kit modals
- `src/actions/` - Modal submission handlers
- `src/services/` - Business logic
- `src/db/` - Database connection and queries
